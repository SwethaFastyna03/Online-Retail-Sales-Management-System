# Online Retail Sales Management System using Oracle SQL

## Project Overview
- This project is a database management system designed to manage customers, products, orders, and sales transactions in an online retail store. It demonstrates database design, data management, and sales analysis using Oracle SQL.

## Objectives
- Manage customer information.
- Store product details and inventory.
- Track customer orders.
- Analyze sales data using SQL queries.
- Generate business reports.

## Tools and Technologies Used
- Oracle SQL
- SQL*Plus
- Relational Database Management System (RDBMS)
  
## Database Tables
- Customers
- Products

## SQL Concepts Used
- DDL Commands (CREATE TABLE)
- DML Commands (INSERT, UPDATE)
- Primary Key
- Foreign Key
- Joins
- Aggregate Functions (SUM, COUNT)
- GROUP BY
- ORDER BY
- Filtering using WHERE clause
## Key Queries Implemented
- Customer Purchase History
- Top Selling Products
- Total Sales Revenue
- Product Inventory Details
- Customer and Order Analysis
- Sample Reports
- Total Revenue Report
- Top Selling Products Report
- Customer Purchase History Report
## Key Learnings
- Designed a relational database schema.
- Established relationships between multiple tables.
- Implemented SQL queries for data retrieval and analysis.
- Generated meaningful business insights from sales data.
## Conclusion

- The Online Retail Sales Management System successfully demonstrates the use of Oracle SQL for managing retail business operations. The project helped in understanding relational database design, table relationships, SQL query writing, and sales data analysis.
